📍 Aqui estão os lugares mais comuns para encontrar ou criar esses badges:

Shields.io → É o serviço mais popular para gerar badges personalizados. Você escolhe o estilo, cor, ícone e texto, e ele gera um link pronto para usar no GitHub.

Badgen.net → Alternativa ao Shields.io, também gera badges rápidos e leves em SVG.

GitHub Actions badges → Muitos projetos usam badges automáticos para mostrar status de build, testes ou cobertura de código.

Docker Hub badges → Para indicar imagens Docker, versão ou status de publicação.

Licença GPL badges → Shields.io já tem modelos prontos para licenças como GPL, MIT, Apache etc.
